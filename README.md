Bitcoin Market Sentiment & Trader Performance Analysis
Project Overview

This project analyzes the relationship between Bitcoin Fear & Greed Index and historical trading performance from Hyperliquid.

The objective was to identify how market sentiment influences trader profitability and build a machine learning model capable of predicting profitable trades.

Key Insights
Greed market conditions generated higher average trader profits.
Winning trade percentages increased during optimistic market sentiment.
Profitability was concentrated among a limited number of cryptocurrencies.
Trading behavior changed significantly over time.
Random Forest successfully classified profitable trades using trading features.
